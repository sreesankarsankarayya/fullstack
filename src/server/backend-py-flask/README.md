# Backend Server-Py

```powershell
python -m venv .e-lit
.e-lit/Scripts/activate
pip install -r requirements-lit.txt
flask run
```

>  Alternatively 

```powershell
./app-lit.py
```

> To detach from the env

```powershell
deactivate
```

> Configuring the environment variables

```powershell
$env:FLASK_RUN_PORT=5001
```
