# Backend Server-Py

```powershell
python -m venv .e-nebula-lit
.e-nebula-lit/Scripts/activate
pip install -r requirements-nebula-lit.txt
flask run
```

> Alternatively

```powershell
./app-nebula-lit.py
```

> To detach from the env

```powershell
deactivate
```

> Configuring the environment variables

```powershell
$env:FLASK_RUN_PORT=5001
```
